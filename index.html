Java

  • It is a high-level programming language developed by James Gosling of Sun Microsystems.
  • It is a platform-independent and object-oriented programming language.
  • Its mainly used for the development of applications.

Variables

Variables are used for storing a value.
A value can be a number, boolean, string or anything.
Eg:
String r
Boolean bl
int a

Constructors

A constructor in Java is a special method that is used to initialize objects.

The constructor is called when an object of a class is created.

Java Constructors cannot be abstract, static, final or synchronized

Methods

Method may look similar to constructors, but, they are different from them.

Eg:
public int add(int a, int b){
return a+b;
}

System.out.print(add(10, 12));

This program when executed, return the sum of the arguments on the console which is 22.
